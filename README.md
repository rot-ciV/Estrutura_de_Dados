# Revisão de Estruturas de Dados

Repositório dedicado à revisão e implementação prática de conceitos fundamentais de Estruturas de Dados. 

Os exercícios e implementações contidos refletem o aprendizado universitário e foram desenvolvidos com base nos roteiros de aula da Universidade Federal de São João del-Rei (UFSJ).

### Módulos de Estudo
- **Módulo I:** Ponteiros e Recursividade
- **Módulo II:** Encapsulamento e Análise de Complexidade
- **Módulo III:** Tipos Abstratos de Dados (TADs)
- **Módulo IV:** Listas

### Tecnologias Utilizadas
- C
- C++
